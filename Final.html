Data Storage

Hexadecimals Notation

Hexadecimal notation, also known as base-16, is a positional numeral system that represents numbers using a radix of sixteen. Unlike the decimal system, which uses ten symbols (0-9), hexadecimal uses sixteen distinct symbols: 0-9 and A-F (o a-f) to represent values from ten to fifteen. This system is widely used in computing and digital electronics because it provides a convenient representation of binary-coded values.

Binary

Binary code is a system of representing text, computer processor instructions, or any other data using a two-symbol system, typically "0" and "1". Each character or instruction is assigned a pattern of binary digits, also known as bits. For example, a binary string of eight bits (a byte) can represent any of 256 possible values.

Database

What is a Database

A database is an organized collection of data that is stored and accessed electronically. It allows users to efficiently store, retrieve, and manage data. Databases are used in a wide range of applications—from websites and mobile apps to enterprise systems and scientific research.

Logic Gates

Logic gates are the fundamental building blocks of digital electronics, performing operations based on Boolean algebra. They take one or more binary inputs (0 or 1) and produce a single binary output. The main types are AND, OR, NOT, NAND, NOR, XOR, and XNOR.

ER Diagram

An Entity-Relationship Diagram (ERD) is a visual representation of the data model for a system, showing entities, their attributes, and the relationships between them. It is widely used in database design to map out the logical structure before implementation.

SQL

A relational database is a collection of information that organizes data in predefined relationships where data is stored in one or maor tables ( or "relations") of columns and rows, making it easy to see and understand how different data structures relate to each other

Programming

Python coding

Python coding refers to the process of writing computer programs using the Python programming language. Python is: High-level – closer to human language than machine code, making it easier to read and write. Interpreted – code runs directly without a separate compilation step. Dynamically typed – you don’t need to declare variable types explicitly. Versatile – supports multiple paradigms: procedural, object-oriented, and functional programming.

Algorithms

What is an Algorithm

An algorithm is a finite sequence of rigorous instructions, used to solve a class of specific problems or to perform a computation.​

Bubble sort

Bubble sort is a sorting algorithm that compares two adjacent elements and swaps them until they are in the intended order.​ Just like the movement of air bubbles in the water that rise to the surface, each element of the array move to the end in each iteration.​

Data Manipulation

What is Data Manipulation

Data manipulation refers to the process of organizing, transforming, and modifying raw data to make it more structured, meaningful, and ready for analysis. It is a critical step in data analysis, data mining, and machine learning, enabling users to derive insights and make informed decisions from unstructured or semi-structured data.

Computer Architecture​

A CPU consists of three parts: the arithmetic/logic unit, which contains the circuitry that performs operations on data (such as addition and subtraction); the control unit, which contains the circuitry for coordinating the machine’s activities; and the register unit, which contains data storage cells (like main memory cells), called registers, that are used for temporary storage of information within the CPU.​
